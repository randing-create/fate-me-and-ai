# 命运 · Fate

一部把贝多芬第五交响曲重新讲成的宇宙寓言：一颗被黑洞裹挟的星球，如何把自己最害怕的那滴暗流，养成了后来照亮宇宙的光。

**概念 · 编排 · 剪辑**：Ran D
**音乐**：L. v. Beethoven, Symphony No. 5 in C minor, Op. 67 — Fulda Symphonic Orchestra / Simon Schindler（[classicals.de](https://www.classicals.de)，CC BY-SA 4.0）
